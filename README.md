# 🌙 Moon2 - Animated Night Sky with Moving Moon

A beautiful Pygame animation featuring a rising moon over mountains, twinkling stars, shooting stars, fireflies, and a gradient night sky.

## ✨ Features

- 🌕 Realistic moving moon with glow effect and detailed craters
- ⭐ 60 twinkling stars
- ☄️ Shooting stars drawn with optimized Bresenham line algorithm
- ⛰️ Layered mountains (back/front)
- 🌲 Shadowed pine trees
- 🛫 Gradient night sky from deep blue to black
- 🪰 Dancing fireflies
- 60 FPS smooth animation
